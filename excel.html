Excel Dashboard

Creating Data Visualizations with Slicers to Identify Trends

Objective

With the given dataset, I wanted to create a dashboard to demonstrate the behavior within the dataset using Excel. Tools utilized within Excel was PivotTables and Slicers.

Cleaning the Data

With the given dataset, I first made sure to remove any duplicates within the dataset.
Initially, terms for Marital Status mentioned "S" and "M" for Single and Married respectively. By selecting the column, clicking CTRL + H, the Find and Replace table allows us to change the terms to clearer terms for the audience to digest. Also, I removed the decimal places from the Income columbn to make it easier to visualize later on.




With the ages varying from 25 to 89, I created an additional column called Age Brackets to create buckets for the ages using the IF statement. This would make it easier to group when visualizing.


Building a Dashboard

PivotTables were utilized to organize selected variables in order to create the visualizations. For example, the relationship between Gender and Income with purchasing a bike was used. This demonstrated that males in general purchase more bikes, especially when making more money.

Bike Purchasing Dashboard

Using the interactive tools, we can see the relationships between people of various education levels, income, and age brackets and their inclination to purchase a bike.
